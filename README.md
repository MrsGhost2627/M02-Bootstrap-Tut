## About This Project

This project was built as part of a Bootstrap 5 tutorial lab. Following along with the Bootstrap 5 Get Started tutorial, I coded a webpage that demonstrates three Bootstrap techniques learned from the tutorial (up through the "Badges" section).

The page is styled around my personal blog concept, **Good Vibes Log** — a fun, positive lifestyle blog.

---

## Three Bootstrap Techniques Used

### 1. Buttons
Bootstrap's button classes (`btn`, `btn-primary`, `btn-outline-dark`, etc.) were used to create styled, consistent call-to-action buttons across the page without writing custom CSS.

### 2. Cards
Bootstrap Cards were used to display blog post previews in a responsive 3-column grid layout. Each card includes a title, description, and a footer button.

### 3. Badges
Bootstrap Badges were used in three ways:
- As category label tags (Lifestyle, Recipes, Wellness, etc.)
- As notification counters on buttons
- As status tags inside a list group (Published, Draft, Coming Soon)
